# API Layer Service

## Usage

Set up go:
```
go get api-layer/cmd
```

Using air in the `api-layer` directory:
```
air .
```

To install [air](https://github.com/air-verse/air):
With Go v1.25+:
```
go install github.com/air-verse/air@latest
```

## Info

This service starts an HTTP server on port `:2745`
